Pi is only in its early stages of development. Many ideas are waiting to happen. If you like this project, you can participate and contribute in a number of ways:

  1. Start using and testing Pi: log in or sign up if you don't have an account yet.

  2. Very important. You did something awesome with Pi? Let us and others know! Post, twitter, tell...

  3. Report bugs in the Pi issue tracker.
    To make things easier, please:
    • Check if the bug is not already reported.
    • If it is a security bug, do not post or discuss publicly about it: please send a message through this contact form.

  4. Help fixing bugs by forking a copy of Pi and pulling requests with bug corrections (even better if you post an issue before).

  5. Another idea? Post a feature request or enhancement in the Pi issue tracker. Or submit a pull request with your new feature. Or contact me.

  6. Write documentation or post tricks about Pi and javascript development with the canvas element.

  7. Enjoy.
